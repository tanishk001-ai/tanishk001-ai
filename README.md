## Featured Project

**[WC Fever 2026](https://github.com/tanishk001-ai/wc_fever_2026)** — [Live Demo ↗](https://wc-fever-2026.vercel.app)  
Full-stack football analytics dashboard built during the actual WC 2026 tournament. Match outcome predictor (XGBoost, 52% test accuracy), xG shot map using StatsBomb open data (~15k shots), and a Monte Carlo group stage tracker (10k sims). Flask + React + Tailwind.
